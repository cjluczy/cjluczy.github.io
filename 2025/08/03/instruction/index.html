开发者必备：Git、Linux 和 Shell 常用指令大全

在日常开发工作中，Git、Linux 和 Shell 是开发者不可或缺的工具。熟练掌握这些工具的常用指令，可以大大提高开发效率。本文将为你详细汇总这些常用指令，帮助你快速查阅和使用。

目录

  1. Git 常用指令
  2. Linux 常用指令
  3. Shell 常用指令

Git 常用指令

基本操作

指令 描述
git init 初始化一个新的 Git 仓库
git clone <repository> 克隆一个仓库到本地
git status 查看当前仓库的状态
git add <file> 将文件添加到暂存区
git add . 将所有变更添加到暂存区
git commit -m "message" 提交暂存区的变更到本地仓库
git push 将本地分支的变更推送到远程仓库
git pull 从远程仓库拉取最新变更到本地
git fetch 从远程仓库获取最新变更，但不自动合并

分支操作

指令 描述
git branch 列出所有本地分支
git branch <branch> 创建一个新的分支
git checkout <branch> 切换到指定分支
git checkout -b <branch> 创建并切换到新分支
git merge <branch> 将指定分支的变更合并到当前分支
git rebase <branch> 对当前分支进行变基操作
git branch -d <branch> 删除本地分支

远程操作

指令 描述
git remote -v 查看远程仓库的 URL
git remote add <name> <url> 添加一个新的远程仓库
git remote remove <name> 删除一个远程仓库
git remote rename <old> <new> 重命名远程仓库

日志和历史

指令 描述
git log 查看提交历史
git log --oneline 简化显示提交历史
git log --graph 以图形方式显示提交历史
git diff 查看暂存区和工作区的差异
git diff <commit> 查看指定提交的差异
git diff <commit1>..<commit2> 查看两个提交之间的差异

其他操作

指令 描述
git reset --hard 重置当前分支到指定状态
git stash 暂存当前工作区的变更
git stash pop 恢复暂存的变更
git tag 创建标签
git tag -a <tagname> -m "message" 创建带注释的标签
git push --tags 推送所有标签到远程仓库

Linux 常用指令

文件和目录操作

指令 描述
ls 列出目录内容
ls -l 以长格式列出目录内容
ls -a 列出所有文件，包括隐藏文件
cd <directory> 切换工作目录
mkdir <directory> 创建新目录
touch <file> 创建新文件
cp <source> <destination> 复制文件或目录
cp -r <source> <destination> 递归复制目录
mv <source> <destination> 移动或重命名文件
rm <file> 删除文件
rm -r <directory> 递归删除目录
rm -f <file> 强制删除文件，不提示确认

文件查看和编辑

指令 描述
cat <file> 查看文件内容
less <file> 分页查看文件内容
head <file> 查看文件的前几行
tail <file> 查看文件的后几行
nano <file> 使用 Nano 编辑器编辑文件
vim <file> 使用 Vim 编辑器编辑文件
sed 's/old/new/' <file> 替换文件中的字符串
grep 'pattern' <file> 在文件中搜索匹配的字符串

系统信息和管理

指令 描述
df -h 查看磁盘使用情况
free -m 查看内存使用情况
top 查看系统进程
ps 查看当前进程
ps aux 查看所有进程
kill <pid> 终止指定进程
kill -9 <pid> 强制终止指定进程
uptime 查看系统运行时间
whoami 查看当前用户
sudo <command> 以超级用户权限执行命令

网络操作

指令 描述
ifconfig 查看网络配置
ping <host> 测试网络连接
netstat -tuln 查看所有开放的端口
curl <url> 下载文件或发送 HTTP 请求
wget <url> 下载文件

权限管理

指令 描述
chmod <permissions> <file> 修改文件权限
chown <user>:<group> <file> 修改文件所有者
ls -l 查看文件权限和所有者

Shell 常用指令

条件语句

if [ 条件 ]; then
    # 条件为真时执行的命令
elif [ 条件 ]; then
    # 条件为真时执行的命令
else
    # 条件为假时执行的命令
fi


### 循环语句
```bash
# for 循环
for i in {1..5}; do
    echo $i
done

# while 循环
while [ 条件 ]; do
    # 循环体
done

###管道和重定向

| 指令                | 描述          |                 |
| ----------------- | ----------- | --------------- |
| `command > file`  | 将命令输出重定向到文件 |                 |
| `command >> file` | 将命令输出追加到文件  |                 |
| `command < file`  | 将文件内容作为命令输入 |                 |
| \`command1        | command2\`  | 将命令1的输出作为命令2的输入 |


其他操作
| 指令       | 描述     |
| -------- | ------ |
| `echo`   | 输出内容   |
| `read`   | 读取用户输入 |
| `exit`   | 退出脚本   |
| `source` | 执行脚本   |
| `export` | 导出环境变量 |
| `alias`  | 创建命令别名 |
